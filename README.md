# Imperium — Conquer the Stars

A turn-based, tick-based strategic galaxy conquest game. Build fleets, colonize planets, and wage war in a dynamic universe. Designed as a browser-based game with a militaristic HUD style, inspired by sci-fi aesthetics.

## Features 
- **Tick-Based Strategic System**: Ships move and battles resolve every 5–10 minutes.
- **Fleet Design & Customization**: Modular ship design with tactical tradeoffs.
- **Colonization & Economy**: Expand your empire by capturing and developing planets.
- **PvP Battles & Diplomacy**: Form alliances, betray rivals, and dominate through strategy.

## Technologies 
- Frontend: HTML, CSS, JavaScript
- Fonts: Orbitron (HUD) and Inter (UI body)
- UI: Chrome + Electric Blue palette, semi-transparent glass panels, starfield parallax background
- Backend (planned): Serverless or dedicated backend for tick processing and game state
